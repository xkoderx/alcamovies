 **To run locally you need an TMDB API key** 
 * And set in /lib/providers/movies_provider.dart
    
## Run Locally 💻

Clone the project

```bash
  git clone https://github.com/xkoderx/alcamovies/
```

Go to the project directory

```bash
  cd alcamovies
```

Install dependencies

```bash
  flutter pub get
```

Run the app

```bash
  flutter run
```
